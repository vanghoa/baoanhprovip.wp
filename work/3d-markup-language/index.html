A fictional markup language for changing the world! This is the final project from the graphic design course in Vietnam - Design 101: The Fundamentals Online 2021.

The project developed a fictional language system controlled by a form of higher being in a four-dimensional space, discovered and utilized by humans. Its structure bears resemblances to markup languages like HTML, Markdown, or XML.

However, unlike digital tools, the language is seamlessly integrated into physical reality. This means that all the scripting materials, platforms, syntaxes, and more already exist in our surroundings. Its purpose is to assist humans in constructing, organizing, reproducing physical, site-specific data, and transforming spiritual, sensory data.

Read more of our document to understand (It’s only available in Vietnamese)

For the presentation of how this language works, I created this demo with Three.js. Visit vanghoa.github.io/3D-Markup-Language/ to see the demo. Creating this experience was quite challenging because I had limited experience with JavaScript at the time. Therefore, it was a steep learning curve, and I had to pick up the skills on the fly.

The language system comprises a set of specialized units: anchor containers, ambient containers and attributes embodied in 3D physical structures. The design of these structures is influenced by the tag elements and attributes commonly found in HTML.

Some sketches by me + 3D prototype by Diệu Ly.
